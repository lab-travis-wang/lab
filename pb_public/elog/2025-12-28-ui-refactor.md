# UI Refactoring: Navigation & Experiment List

为了提升移动端体验和整体视觉的精致度，今天对导航栏和实验列表进行了两项主要重构。

## 1. 紧凑型导航栏 (Compact Navbar)

原有的导航栏在移动端容易换行，且中英文切换占据了较多空间。

**改进点：**
- **单行布局**：强制导航栏内容在同一行显示，中间的导航链接部分在小屏幕下支持横向滚动 (Horizontal Scroll)。
- **下拉菜单语言切换**：将 `EN / 中文` 的文本切换改为 `<select>` 下拉菜单，大大节省空间。
- **GitHub 图标组合**：将 GitHub 链接简化为图标，并与语言切换组合在最右侧，作为固定的功能区。

## 2. 实验列表卡片 (Experiment Cards)

实验列表原本的设计较为松散，且包含独立的"详情"按钮。

**改进点：**
- **整体点击**：去掉了冗余的"详情"按钮，现在整个卡片区域均可点击跳转。
- **Logo/Icon 支持**：
  - 优先显示 `logo` (图片)。
  - 其次显示 `icon` (Emoji)。
  - 均无则显示默认 Emoji `🧪`。
- **布局优化**：
  - 左侧：实验状态、名称、描述。
  - 右侧：**实验编号 (#001)** 以大号灰色字体显示在 **Logo** 正上方，保持视觉平衡且保留了实验室的"编号感"。
